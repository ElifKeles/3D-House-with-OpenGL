To run this file: (compile and run as)
		make
		./program 

-> Rotation: Use up down left and right arrow keys to rotate the house.

To make scaling and transformation work, rotation needs to be stopped.
Press s to stop/ unstop. 

-> Translation: Press l r d u keys to move left right down and up accordingly.

-> Scaling: Press a to minimize A to maximize the house.

To quit the running program, press q or ESC key.
